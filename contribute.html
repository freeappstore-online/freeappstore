Improve existing apps

Every app is open source on GitHub. Fork, fix bugs, add features, submit a PR. We review within 48 hours.

Step by step

  1. Clone the template
    git clone https://github.com/freeappstore-online/template-standalone my-app

    For games, use game templates (canvas, grid, cards, 3D).

  2. Build your app

    Edit web/src/App.tsx. The Shell component gives you the sidebar layout, fonts, and dark mode for free. Use pnpm dev for hot reload.

  3. Submit for review

    Push your code to a public GitHub repo and open a submission. Include your repo link and a brief description.

  4. Go live

    After approval, we provision your app — GitHub repo in our org, custom domain, auto-deploy. Your app is live at yourapp.freeappstore.online. Every push to main auto-deploys.

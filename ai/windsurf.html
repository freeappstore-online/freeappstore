Windsurf on FreeAppStore

AI code editor by Codeium. Scaffold locally, add platform context, prompt the build.

Setup

  1. Install FreeAppStore CLI
    npm i -g @freeappstore/cli && fas login
  2. Scaffold your app
    fas init my-app && cd my-app && pnpm install

Build

Open in Windsurf and prompt: "Build me a flashcard app using the Shell layout and brand tokens."

Give it context

Add to your Windsurf global rules or .windsurfrules:

Read https://freeappstore.online/skills.md for platform skills, brand mandates, compliance rules, and the publish flow.

Publish

  1. Check compliance
    fas check
  2. Publish to the store
    fas publish

    Creates repo, hosting route, custom subdomain, and store listing — all at once.

  3. Future updates
    git push origin main

    Auto-deploys in ~30 seconds.

Add user accounts & cloud storage

Standalone apps use localStorage. If you need GitHub sign-in, per-user cloud storage, realtime rooms, or a secret-injecting API proxy:

cd web && pnpm add @freeappstore/sdk
  • Auth — GitHub OAuth. fas.auth.signIn()
  • KV — Per-user storage. fas.kv.set('key', value)
  • Rooms — Realtime WebSocket. fas.rooms.join('lobby')
  • Proxy — Server-side API keys. fas.proxy.fetch('api.example.com/data')

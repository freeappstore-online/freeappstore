MCP Server

Connect your AI agent to FreeAppStore. List apps, check deploys, get SDK docs — all as tools your agent can call directly.

What is MCP?

Model Context Protocol (MCP) is an open standard that lets AI assistants call tools on remote servers. Instead of copy-pasting docs or running CLI commands, your AI agent can query the FreeAppStore API directly.

Connect

Add this to your AI tool's MCP config:

Claude Code

Add to ~/.claude.json:

{
  "mcpServers": {
    "freeappstore": {
      "command": "npx",
      "args": ["mcp-remote", "https://mcp.freeappstore.online/mcp"]
    }
  }
}

Cursor

Settings → MCP → Add Server:

Name: FreeAppStore
Command: npx mcp-remote https://mcp.freeappstore.online/mcp

Any MCP-compatible client

Endpoint: https://mcp.freeappstore.online/mcp (Streamable HTTP transport)

Tools

ToolAuthDescription
deploy_status None Check the last 5 GitHub Actions deploy runs for any app. Shows pass/fail, commit SHA, and timestamp.
app_info None Get live URL, repo URL, store listing URL, and current up/down status for any app.
sdk_reference None Quick SDK reference with import paths and usage examples. Filter by feature: auth, kv, counters, collections, rooms, proxy, hooks, ui.
platform_guide None Fetch the full SKILLS.md — the complete platform guide covering tech stack, project structure, SDK, CLI, compliance, and deploy flow.
list_apps FAS token List your published apps with categories, descriptions, and URLs. Pass your fas login session token.

Example usage

Once connected, your AI agent can do things like:

  • "Check if my last deploy succeeded" → calls deploy_status
  • "How do I use KV storage?" → calls sdk_reference with feature: "kv"
  • "Is timer.freeappstore.online up?" → calls app_info
  • "What's the platform tech stack?" → calls platform_guide

Discovery

The MCP server is auto-discoverable at freeappstore.online/.well-known/mcp.json.

Source

The MCP server is open source: github.com/freeappstore-online/mcp
